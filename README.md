# 🧠 Análise Deep Learning 

## 📖 Sobre o projeto

Este repositório contém um relatório acadêmico desenvolvido para a disciplina de Inteligência Artificial.

O trabalho apresenta conceitos de Machine Learning e Deep Learning, abordando redes neurais artificiais, aprendizado supervisionado, overfitting, interpretabilidade dos modelos e um estudo de caso da plataforma NeuroLearn.

---

## 📚 Conteúdo

- Machine Learning
- Deep Learning
- Redes Neurais Artificiais
- Aprendizado Supervisionado
- Overfitting
- Interpretabilidade (Explainable AI)
